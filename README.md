**SimpleMace** - plugin that allows you to customize the process of obtaining a mace and its quantity on the server. Supported for Paper, Bukkit, Spigot, Purpur.

---

**Features**
- Flexible customization of receiving and interacting with the mace.
- Support for HEX colors.

---

**Command list**
- /simplemace (/mc) reload - Reload plugin configuration.
- /simplemace (/mc) list - List of players with maces.

**Permission list**
- simplemace.use - Rights to use the /simplemace command.
- simplemace.reload - Rights to reset the configuration.
